# card/+page.svelte

Demonstrates the FeatureDemoCard animated card component from Aceternity UI.

```svelte
<script>
  import FeatureDemoCard from "$lib/aceternity-ui/components/cards/feature-block-animated/FeatureDemoCard.svelte";
</script>

<div class="flex justify-center items-center h-screen">
  <FeatureDemoCard />
</div>
```

- Shows a centered animated feature card.
